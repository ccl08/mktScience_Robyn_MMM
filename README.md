# Marketing Mix Modeling with Robyn
This project implements a Marketing Mix Model (MMM) using the Robyn framework to estimate the impact of different marketing channels on business outcomes.

The goal is to understand how media investments contribute to conversions and revenue, enabling data-driven budget allocation and marketing optimization.

#Project Overview

Marketing Mix Modeling helps answer key questions:
	•	Which marketing channels drive the most incremental value?
	•	What is the optimal allocation of marketing budget?
	•	How does diminishing return affect media spend?

This project uses Robyn, Meta’s open-source MMM package in R, to build and evaluate a marketing attribution model.

#Tools & Technologies
	•	R
	•	Robyn (Meta MMM framework)
	•	Marketing Mix Modeling
	•	Time-series regression

	•	Saturation modeling
